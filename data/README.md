# Data: real-photometric-noise experiment

Inputs and Phase-0 products for the real-noise + Lipschitz study
(`paper/paper.tex`). Only files used by this experiment are kept.

## Source catalogues

| File | Size | Source | Description |
|------|------|--------|-------------|
| `star_classification.csv` | 17 MB | [Kaggle/SDSS DR17](https://www.kaggle.com/datasets/fedesoriano/stellar-classification-dataset-sdss17) | 100,000 SDSS objects (GALAXY/STAR/QSO). Provides `obj_ID`, `u,g,r,i,z`, `redshift`, `class`. The `obj_ID` is the join key to SkyServer `PhotoObjAll` for magnitude errors. |

## Phase-0 products (per-object photometric uncertainties)

| File | Generated by | Description |
|------|--------------|-------------|
| `sdss_errors.csv` | `scripts/fetch_sdss_errors.py` | `objID` + `psfMagErr_{u,g,r,i,z}` from SkyServer `PhotoObjAll`. 60,411 rows = unique matched `objID`s (of 78,053 unique in the catalogue; the remaining 17,642 have no `PhotoObjAll` match). Joined back onto the 100,000-row catalogue these cover 76,823 rows (76.8%), of which 73,302 remain usable after dropping sentinel/non-positive errors. |
| `desi_dr1_sample_with_errors.csv` | `scripts/extract_desi_errors.py` | Balanced 99,999-object DESI DR1 sample: `class, redshift, g, r, z_mag, bp, rp` + `g_err, r_err, z_err` (σ from FLUX_IVAR). |

## DESI FITS source files (large, external, not committed)

Downloaded once from DESI DR1; cached locally and read by `extract_desi_errors.py`.

| File | Size | URL |
|------|------|-----|
| `zpix-main-bright.fits` | 8.8 GB | `https://data.desi.lbl.gov/public/dr1/spectro/redux/iron/zcatalog/v1/zpix-main-bright.fits` |
| `zpix-main-dark.fits` | 10.5 GB | `https://data.desi.lbl.gov/public/dr1/spectro/redux/iron/zcatalog/v1/zpix-main-dark.fits` |

## Integrity

`checksums.sha256` covers the three CSV data files (not the multi-GB FITS).
Verify with `sha256sum -c checksums.sha256`.
